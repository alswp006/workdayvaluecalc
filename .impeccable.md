# .impeccable.md

## Design Context

### Users
직장인 20-40대가 구매 전 '이 물건이 내 몇 시간 노동값인가'를 빠르게 판단하려고 사용합니다. 절약 습관을 기르거나 FIRE에 관심 있는 금융 리터러시 높은 사용자들이 게임처럼 즐기며 소비 의사결정을 합리화하는 것이 주요 목표입니다.

### Brand Personality
**게임화된 금융**, **명확한 가치판단**, **공유와 도전**. 토스의 친근하고 직관적인 톤을 유지하되, 소비 결정의 무게감을 재미있게 프레이밍합니다. 죄책감보다는 '현명한 선택'의 희열을 전달하는 밝고 격려적 톤입니다.

### Aesthetic Direction
TDS (Toss Design System) 준수. @toss/tds-mobile 컴포넌트만 사용. 커스텀 비주얼 방향 없음.

### UX Principles
1. **입력 최소화**: 연봉 1회 입력 후 localStorage 저장으로 반복 사용 시 진입장벽 제거  
2. **즉시 피드백**: 가격 입력 → 노동시간 애니메이션 (분급 환산) 즉시 표시로 소비 심리 자극  
3. **공유 가능성**: 결과 이미지 생성 및 친구 비교 기능으로 바이럴 루프 설계  
4. **재방문 유인**: 일일 알림 및 누적 소비/절약 추적으로 일회성 사용 방지  
5. **간결성 우선**: 핵심 계산(시급·분급·초급)만 명확히, 복잡도 최소화
