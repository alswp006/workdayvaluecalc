# WorkdayValueCalc

## Platform
앱인토스 (Vite + React + TDS)

## One-liner
구매 전 '이 물건은 내 몇 시간 노동값인가'를 재미있게 계산해주는 시간-가치 계산기

## Problem
월급 300만원 직장인이 6만원 신발을 살 때 '이게 비싼 건지' 직관적으로 모른다. 시간으로 환산하면 소비 결정이 명확해지고, 게임처럼 즐기면서 절약 습관을 기를 수 있다.

## Target Users
월급쟁이 직장인(20-40대), 소비 전 '이게 맞는 선택인가' 고민하는 사람, 절약·FIRE 관심자, 게임화된 금융 도구를 즐기는 MZ세대

## Core Features
1. 연봉/월급 입력 → 시급·분급·초급 자동 계산
2. 바코드/사진 스캔으로 실시간 상품 가격 인식 후 '몇 시간 일해야 하는지' 즉시 표시
3. 소비 금액 입력 시 '분급 환산' 애니메이션 재생 (노동 시간을 시각적·재미있게 표현)
4. 고정지출(월세, 구독료 등) → 매달 며칠치 노동인지 시각화
5. 목표 저축액까지 몇 달/일 노동 필요 계산
6. 결과 이미지 공유 기능 + 친구 도전 비교 (바이럴)

## Monetization
- Model: ads + gamification engagement
- Estimated MRR: 4500000
- Reasoning: 리워드 광고: DAU 4,500 × 전환율 25% × 30일 × CPM $4/1000 × 1350 × 0.85 ≈ 180만원 (게임화로 인게이지먼트 증가). 배너 광고: DAU 4,500 × 2.5회 × 30일 × CPM $1/1000 × 1350 × 0.85 ≈ 300만원. 바코드/이미지 스캔 프리미엄 기능 구독: DAU 4,500 × 전환율 3% × 월 4,900원 ≈ 70만원. 공유 기능과 친구 비교로 바이럴 가속, DAU 증대 예상.

## Existing Alternatives
- 일반 시급 계산기 앱: 시급 계산 기능 유사 → Our differentiation: 소비 결정 연동 + 노동시간 환산 프레이밍이 핵심. 단순 계산기가 아닌 소비 철학 도구.
- 토스 소비 내역: 소비 분석 → Our differentiation: 토스는 사후 분석, 본 앱은 소비 전 의사결정 지원

## Risks
### limitations
- 계산 결과가 단순해 반복 사용 동기 약함
- 입력 데이터(연봉)를 매번 입력해야 하는 불편함 (localStorage로 저장 가능하나 초기 이탈 위험)
### challenges
- 일회성 사용 후 재방문 유인 부족 — 일일 '오늘의 소비 환산' 알림 기능 필요
- 결과가 단순명료해 리워드 광고 게이팅 설득력 낮을 수 있음
### marketRisks
- 유사한 단순 계산기 앱이 앱인토스에 이미 존재할 가능성
- 소비 결정 도구 특성상 감성적 반감(죄책감 유발) 가능성
